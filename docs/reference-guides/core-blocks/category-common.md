# Common Blocks

- [Form](https://developer.wordpress.org/block-editor/reference-guides/core-blocks/core-blocks-common/core-block-form/) — `core/form`: A form.
- [Input Field](https://developer.wordpress.org/block-editor/reference-guides/core-blocks/core-blocks-common/core-block-form-input/) — `core/form-input`: The basic building block for forms.
- [Form Submission Notification](https://developer.wordpress.org/block-editor/reference-guides/core-blocks/core-blocks-common/core-block-form-submission-notification/) — `core/form-submission-notification`: Provide a notification message after the form has been submitted.
- [Form Submit Button](https://developer.wordpress.org/block-editor/reference-guides/core-blocks/core-blocks-common/core-block-form-submit-button/) — `core/form-submit-button`: A submission button for forms.
